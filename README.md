# 👨🏽‍💻 Pedro Pablo

**`Desenvolverdor,Analista e Eng. de Computação`**

Me chamo Pedro Pablo, tenho 23 anos e sou natural de Cametá. Concluí o ensino médio no Colégio Atitude. Atualmente, estou cursando Engenharia de Computação na Universidade Federal do Pára (UFPA). Sou apaixonado por tecnologia e compartilho aqui meus conhecimentos e avanços durante minha tragetória.

Possuo experiência com diversas tecnologias modernas, incluindo desenvolvimento web com HTML, CSS e JavaScript, além de frameworks como React. Também trabalho com linguagens como Java, C e Python, aplicando conceitos de programação orientada a objetos, análise de dados e desenvolvimento de sistemas.

Tenho familiaridade com bancos de dados, versionamento com Git e integração de sistemas, o que me permite atuar de forma completa no ciclo de desenvolvimento de software. Estou constantemente aprendendo e evoluindo, explorando novas ferramentas e boas práticas para melhorar a qualidade do código e das soluções que desenvolvo.

Meu objetivo é crescer profissionalmente contribuindo com projetos inovadores, sempre buscando excelência técnica e impacto positivo através da tecnologia.
